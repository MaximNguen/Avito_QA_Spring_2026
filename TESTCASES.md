# TESTCASES.md
# Тест-кейсы для API Авито

## Обзор
Документ содержит тест-кейсы для API-эндпоинтов сервиса Авито.

## Структура тест-кейса
Каждый тест-кейс включает:
- **ID**: Уникальный идентификатор
- **Название**: Описание тест-кейса
- **Приоритет**: Высокий/Средний/Низкий
- **Тип**: Позитивный/Негативный/Граничный
- **Предусловия**: Необходимые настройки
- **Шаги**: Шаги выполнения теста
- **Ожидаемый результат**: Ожидаемый результат выполнения

## Используемые техники тест-дизайна
- **Классы эквивалентности**: валидные/невалидные значения sellerID, price, name
- **Анализ граничных значений**: границы sellerID (111111, 999999)
- **Попарное тестирование**: комбинации полей
- **State-transition testing**: состояния объявления (создано → получено → удалено)
